Privacy Policy for YouTube Caption Search

Effective date: 2026-05-21

YouTube Caption Search is designed as a local-first browser extension.

Data collection

YouTube Caption Search does not collect personal information.

The extension does not collect, transmit, sell, rent, or share user data with third parties.

Data stored locally

The extension may store the following data locally in your browser:

  • Saved timestamped moments for YouTube videos
  • Caption snippet context preference, such as Short, Medium, or Long
  • Cached caption/transcript data for recently viewed videos
  • Temporary search result state while the extension side panel is open

This data is stored locally using browser storage mechanisms such as localStorage.

Data transmission

The extension does not send saved moments, searches, captions, transcripts, or preferences to any external server controlled by the extension developer.

The extension interacts with the currently open YouTube page in order to read caption metadata, load available captions, and jump the video player to selected timestamps.

Third-party services

The extension runs on YouTube pages and interacts with YouTube-provided caption or transcript data that is available to the active page.

Use of YouTube is governed by YouTube’s own terms and privacy policies.

Permissions

The extension requests permissions needed to:

  • Open a side panel
  • Read the active YouTube video page
  • Execute scripts on YouTube pages to access caption data and control video seeking
  • Identify the active tab and current YouTube video URL

The extension does not request permissions for unrelated websites.

Clearing local data

Users can clear saved moments and cached captions through the extension UI.

Users may also remove all extension data by uninstalling the extension or clearing browser site/extension storage.

Contact

For privacy questions, contact the extension developer through the support contact provided on the Chrome Web Store listing.
